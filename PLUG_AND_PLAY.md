# PLUG_AND_PLAY — tiny-neural

> *Integration guide.*


## Feature Flags

| Feature | Default | Description |
|---------|---------|-------------|
| `std` | yes | Standard library |
| `alloc` | yes | Allocator support |

## Integration

Import `tiny_neural` in your project to access the functionality.

```
use tiny_neural::{};
```

## Compatibility

- **Rust edition**: 2021+
- **Targets**: All tier-1 Rust targets
